# PSI_Flock
Pencarian flock dengan menggunakan algoritma Plane Sweep, Binary Signature, dan Inverted Index. Dibuat sebagai syarat tugas akhir.

# Referensi Utama
Tanaka, P. S., Vieira, M. R., dan Kaster, D. S. (2016) An improved base algorithm for online discovery of flock patterns in trajectories. *Journal of Information and Data Management*, 7, 52â€“67

Maven minimal versi -3.6.0
